## Infomation

> <Highlight color="#00ccff"> GET </Highlight>

> <Highlight color="#ff660f"> Array </Highlight>

> <Highlight color="#c1c10f"> Object </Highlight>

予約されているアイテムの情報です。

返す値は `ReserveItem[]` です。

## Properties [Partial]

### 予約されているアイテムがない場合

> <Highlight color="#c1c10f"> Object </Highlight>

| 値 | 型 | いつももあるか | 説明 |
| --- | --- | --- | --- |
| err | String | YES | エラー内容です。 |

### 予約されているアイテムがある場合

| 値 | 型 | いつももあるか | 説明 |
| --- | --- | --- | --- |
| items | Array | YES | 予約全データです。下のデータが配列として保管されています。 |


> <Highlight color="#ff660f"> Array </Highlight>

| 値 | 型 | いつももあるか | 説明 |
| --- | --- | --- | --- |
| Items | ItemObject | might be | 予約されているアイテムの数、名前が保管されています。 |
| ReserveNo | String | YES | 予約番号です。全角文字・日本語は許されていません。 |
| AllPrice | Number | YES | 予約総額が記録されています。 |
| events | Events[] | might be | イベントカテゴリの予約データが保管されています。|
| date | Date | YES | 追加日です。たぶん UTC+8 です。|
