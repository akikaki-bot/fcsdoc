## Welcome

やぁ。管理するあなたには！！とっても感謝感激パーリナイで感謝しています。

とりあえずありがとうってことです。

まぁ、早速管理方法について伝授しちゃいますよ💕

## Step 1 - お店を展開しよう。

### まずはいたずら防止

```
/kanri
```


にアクセスをします。

そしたら**注文の停止**っていう感じのボタンを押して予約を停止しましょう。

いたずらの防止です。

### 在庫をまずセットしよう。

```
/kanri/stock
```

にアクセスをします。

初期在庫ボタンをクリックして有効にし、今ある在庫をセットします。

:::info
** ちなみに **<br />
`reserve.json`っていう管理ファイル内のアイテムしか追加できないです。<br />
なのでもしアイテムを追加する場合は管理ファイル`reserve.json`をSSHからいじってください。
:::

### お店を展開する場所へ行き、管理画面へアクセスしよう。

```
/reserveKanri
```

にアクセスをします。

そしたら自分が担当するテーブル番号を決め準備完了！

あとは来るまで待ってましょう。

### お客さんが分かるようにしよう

```
/calling
```

にアクセスをします。

そしたら呼ばれる人が分かるようになります。

:::info
** ちなみに **<br />
大画面なやつがあった方がいいかも。iPad推奨だけどね（矛盾）
:::

### 迎え入れよう

```
/kanri
```

にもう一度アクセスをして、注文受け入れを開始しよう！

がんばって！

## Step2 - 注文を受け入れよう

```
/reserveKanri
```

担当者のあなた！

ここではそんな感じの操作方法が分かるようになってるよ。おめでとう。

### 注文が来たら

その準備をしよう。

注文が準備できたテーブルから、呼び出しボタンを押して迎え入れよう。

### お客さんが来たら？

呼び出し停止ボタンを押して、アイスづくりを開始しよう。

:::warning
**ちなみに**<br />
注文の取り合いにならないようにね！
:::

## Step3 - お金を見よう。

```

```

担当者や最終売上金額がきになるそこのあなた！

このページをみてくれませ。

